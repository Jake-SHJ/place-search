- 사용자(user)
    - id, password를 상태 값으로 가진다.
    - password는 암호화 한다.
    - 정보를 비교하여 로그인 여부를 판단 할 수 있다.
- 장소(place)
    - 검색한 장소들의 정보를 가지고 있다.
    - 한 장소에 대한 정보를 가지고 있다.
    - 정보를 보여 준다.
- 검색(search)
    - 검색을 할 수 있다.
    - 키워드를 상태 값으로 가진다.
    - 상태 값으로 외부 API 를 통해 데이터를 가져올 수 있다.
